Descripción:
Se trata de un programa que espera recibir por UART 2 datos correspondientes al valor que entregaría un sensor de color RGB y mediante una Red Neuronal Perceptron se implementa la identificación de color.
Los datos RGB se setean a través del array de funciones implementado.
